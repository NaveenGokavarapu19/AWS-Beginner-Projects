# How to Use the Party Rock Project

The Party Rock project is designed to help you create beginner-friendly cloud projects in under 30 minutes. Follow the steps below to get started:

## Step 1: Select a Cloud Provider
Currently, the Party Rock project supports AWS. Future updates may include other cloud providers.

## Step 2: Choose a Programming Language
Select the programming language you want to use for your project. Supported languages include:
- Python
- JavaScript
- Java
- Go

## Step 3: Generate a Beginner Project
Run the Party Rock tool to generate a project template. The tool will create a simple project with detailed instructions to help you complete it.

## Step 4: Follow the Instructions
Each generated project comes with step-by-step instructions. These instructions will guide you through:
1. Setting up your environment.
2. Writing the code.
3. Deploying the solution to the cloud.

## Step 5: Watch the Usage Video
For a visual demonstration, refer to the usage video included in the repository. The video will walk you through the process of using the Party Rock tool.
